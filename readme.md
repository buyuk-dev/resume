# Resume

My resume, maintained in Markdown with LaTeX formatting for PDF generation.

## Files

- **[source.md](source.md)** - Source file (Pandoc Markdown + LaTeX)
- **[MichalskiMichal.pdf](MichalskiMichal.pdf)** - Generated PDF

## Building

Requires `pandoc` and `xelatex`:

```bash
bash build.sh
```

The PDF is automatically rebuilt via GitHub Actions on push.
